# PharmacyApp
###  Опис проєкту
Програмне забезпечення призначене для автоматизації робочого місця провізора та завідувача аптеки. Система дозволяє вести облік медикаментів, контролювати терміни придатності та здійснювати швидкий продаж товарів.

###  Технічний стек
*   **Мова програмування:** C#
*   **Платформа:** .NET Framework / Windows Forms
*   **База даних:** MS SQL Server (LocalDB)
*   **Технологія доступу до даних:** ADO.NET

###  Функціональні можливості
1.  **Роль "Провізор":**
    *   Пошук ліків за назвою.
    *   Формування чека та продаж.
    *   Автоматична перевірка терміну придатності при продажу.
2.  **Роль "Адміністратор":**
    *   Оприбуткування товару (прихідна накладна).
    *   Перегляд та керування складськими залишками.

###  Як запустити
1.  Клонуйте репозиторій або завантажте архів.
2.  Відкрийте файл `PharmacyApp.sln` у Visual Studio 2022.
3.  Переконайтеся, що встановлено компонент **"Зберігання та обробка даних" (SQL Server LocalDB)**.
4.  Запустіть проєкт (F5). База даних ініціалізується автоматично (або скористайтеся скриптом `script.sql` за необхідності).
